# 組織・業務課題一覧

Dental X-Ray Navigator が解決すべき3つの問題と、現状のギャップ。

---

## アプリの目的

| 項目 | 内容 |
|------|------|
| **解決したい問題** | ①撮影待ち患者管理 ②照射録記録 ③保険点数集計 |
| **主なユーザー** | 歯科診療放射線技師、歯科衛生士 |
| **運用形態** | 1医院の内製ツール |

---

## 1. 撮影待ち患者管理

### 解決したかった問題
- 誰がどの撮影を待っているか分からない
- 依頼の抜け漏れが発生する
- 患者の待ち時間が長くなる

### 現状の実装
| 機能 | 状態 | 備考 |
|------|------|------|
| 依頼一覧表示（タスク画面） | ✅ 実装済 | pending状態をカード表示 |
| 患者情報の自動入力 | ✅ 実装済 | ID入力でDB検索 |
| 撮影種別の複数選択 | ✅ 実装済 | 7種類対応 |
| 予約日時の指定 | ✅ 実装済 | scheduledDate/Time |
| 移動先の指定 | ✅ 実装済 | locationTo |

### 残っている課題

| 課題 | 優先度 | 詳細 |
|------|--------|------|
| **リロードで見失う** | 高 | ページ更新でログアウト、タスク画面に戻れない |
| **撮影順序が不明** | 中 | 複数依頼があるとき、どれを先にやるべきか分からない |
| **緊急度の表示なし** | 中 | 急ぎの撮影を区別できない |
| **通知機能なし** | 低 | 新しい依頼が来ても気づかない |

---

## 2. 照射録の記録・管理

### 解決したかった問題
- 紙での記録が煩雑
- 後から照射条件を探しにくい
- 患者ごとの履歴が追えない

### 現状の実装
| 機能 | 状態 | 備考 |
|------|------|------|
| 照射条件入力（kV/mA/sec） | ✅ 実装済 | 種別ごとに記録 |
| 被曝テンプレート自動適用 | ✅ 実装済 | 年齢・体格から初期値設定 |
| 担当者名の記録 | ✅ 実装済 | operatorName |
| 撮影完了履歴 | ✅ 実装済 | history画面 |

### 残っている課題

| 課題 | 優先度 | 詳細 |
|------|--------|------|
| **患者別の履歴が見れない** | 高 | 特定患者の過去の撮影を一覧できない |
| **照射録の検索機能なし** | 高 | 日付・患者名での絞り込みができない |
| **照射録の印刷/出力なし** | 中 | 紙での保管が必要な場合に対応できない |
| **撮影画像との紐付けなし** | 低 | PACSとの連携は未対応 |

---

## 3. 保険点数の集計

### 解決したかった問題
- 手計算で時間がかかる
- 計算ミスが発生する
- 日次・月次の集計が大変

### 現状の実装
| 機能 | 状態 | 備考 |
|------|------|------|
| 依頼時の点数自動計算 | ✅ 実装済 | 種別ごとの基本点数 |
| BITEWING左右の乗算 | ✅ 実装済 | 48点×選択側数 |
| 本日の点数合計 | ✅ 実装済 | StatsDashboard |
| 撮影種別シェア | ✅ 実装済 | 円グラフ |

### 残っている課題

| 課題 | 優先度 | 詳細 |
|------|--------|------|
| **期間指定の集計なし** | 高 | 月次集計ができない |
| **CSV/PDF出力なし** | 高 | 会計・経理への報告ができない |
| **点数マスタの更新** | 中 | 診療報酬改定時に手動でコード修正が必要 |
| **レセコン連携なし** | 低 | 手入力での転記が必要 |

---

## 優先度サマリー

### 高優先度（業務に直接影響）
1. セッション永続化（リロードで見失う問題）
2. 患者別の撮影履歴表示
3. 照射録の検索機能
4. 期間指定の点数集計
5. CSV/PDF出力

### 中優先度（効率化）
6. 撮影順序・緊急度の表示
7. 照射録の印刷機能
8. 点数マスタの管理画面

### 低優先度（将来的な拡張）
9. 通知機能
10. PACS連携
11. レセコン連携

## 現場ユーザーの視点チェックリスト

### 技師・衛生士が毎日使う上で
- [ ] ログインが面倒でないか？
- [ ] タスク一覧は見やすいか？
- [ ] 照射条件の入力は素早くできるか？
- [ ] 過去の記録をすぐ探せるか？
- [ ] 点数集計は正確か？

### 管理者が月次で確認する上で
- [ ] 月間の撮影件数は把握できるか？
- [ ] 月間の保険点数は集計できるか？
- [ ] 担当者別の実績は確認できるか？
